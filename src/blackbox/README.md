## Installation

1. Copy the `Everblush.json` file from this folder and paste in `~/.var/app/com.raggesilver.BlackBox/data/blackbox/schemes`.
2. Open Black Box, click on `Preferences > Terminal`, go to `Theme` and select `Everblush` theme from available options.
